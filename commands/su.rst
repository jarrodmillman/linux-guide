.. _command-su:

su
==

Name
----

su -- run a shell with substitute user and group IDs

Synopsis
--------

**su** [option...]

Description
-----------

Change the effective user id and group id to that of USER.

Frequently used options
-----------------------

-, -l, --login 
    make the shell a login shell

-c, --command=COMMAND 
    pass a single COMMAND to the shell with -c
