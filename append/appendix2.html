Directory Description

Table B-1. FMRI specific Filesystem Hierarchy

Directory Description
/r Automount mount point for all remote filesystems
/r/d0 Data directories
/r/d1 Data directories
/r/fmri top-level for shared fmri resources
/r/fmri/bin shared fmri stuff binaries
/r/fmri/lib shared fmri 32bit libraries
/r/fmri/lib64 shared fmri 64bit libraries
/usr/fmri symbolic link to /r/fmri
